# Windborne Embedded Electrical Engineer Challenge Submission

## Project
For this project, I developed a PCB with 7 flickering LEDs to create a backlit board. My goal was to create the effect of a haunted house with flickering lights inside as a Halloween decoration.

## Files
pcb_schematic.pdf: This file shows the circuit schematic for my PCB.
pcb_routing.png: This file shows the routing layout for my PCB, along with a description of one challenge that I solved.
halloween_pcb.jpeg: This file is an image of my PCB built.
